# לוח מודל עסקי — עוז, נעמה, ערן

אפליקציה שיתופית בזמן אמת. כל שינוי נשמר אוטומטית ומסתנכרן לכולם תוך 3 שניות.

---

## פריסה ב-5 דקות

### שלב 1 — GitHub

1. היכנסו ל-[github.com](https://github.com) וצרו חשבון (אם אין לכם)
2. לחצו על **New repository**
3. שם: `business-model-canvas`
4. לחצו **Create repository**
5. גררו את כל תיקיית הפרויקט לחלון שנפתח (או השתמשו ב-Upload files)

### שלב 2 — Vercel

1. היכנסו ל-[vercel.com](https://vercel.com) וצרו חשבון (התחברו עם GitHub)
2. לחצו **Add New → Project**
3. בחרו את ה-repository שיצרתם
4. לחצו **Deploy** — Vercel יזהה הכל אוטומטית

### שלב 3 — מסד נתונים (Vercel KV)

1. בדשבורד של Vercel, היכנסו לפרויקט שלכם
2. לחצו על **Storage** בתפריט העליון
3. לחצו **Create Database → KV**
4. תנו לו שם כלשהו ולחצו **Create**
5. לחצו **Connect to Project** ובחרו את הפרויקט שלכם
6. חזרו ל-**Deployments** ולחצו **Redeploy** (פעם אחת בלבד)

### זהו!

Vercel ייתן לכם URL בצורת `your-project.vercel.app` — שתפו אותו עם עוז, נעמה וערן.

---

## שימוש

- בחרו את שמכם בסרגל העליון
- הוסיפו רעיונות לכל תא
- לחצו **קישור בין רעיונות** לחיבור בין רשומות
- כל שינוי נשמר ומופיע לשאר הצוות תוך ~3 שניות
